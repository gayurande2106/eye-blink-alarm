# 👁️ AI Eye Blink Alarm

An AI-powered drowsiness detection system that monitors eye closure using MediaPipe Face Mesh and triggers an alarm if the eyes remain closed for more than 5 seconds.

## 🚀 Features

- Real-time webcam monitoring
- Eye Aspect Ratio (EAR) calculation
- Detects prolonged eye closure
- Alarm sound without external audio file
- Live status overlay
- Progress bar and alert display

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Pygame

## 📦 Installation

```bash
pip install opencv-python mediapipe pygame numpy
```

## ▶️ Run

```bash
python eye_blink_alarm.py
```

## 📸 Output

- Detects face
- Tracks both eyes
- Triggers alarm after 5 seconds of closed eyes
